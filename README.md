# Verta
A simple minimal watch face for Tizen 2.3 or newer

## Requirements
 - Gear Watch Designer 1.6.2 (with 32bit jre 1.8 for some reason)
 - Installed `Barlow Semi Condensed` in `C:\Program Files (x86)\GearWatchDesigner\font`
